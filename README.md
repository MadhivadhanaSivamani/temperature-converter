The main objective of this project is to:

Convert temperatures accurately between Celsius, Fahrenheit, and Kelvin.
Provide an easy-to-use graphical interface instead of using the command line.
Demonstrate the use of Python GUI programming, event handling, and mathematical calculations.
Technologies Used
Programming Language: Python
GUI Library: CustomTkinter
Additional Module: Tkinter MessageBox (for error messages)
How the Application Works
User Input
The user enters a temperature value into the input box.
Select Temperature Unit
The user chooses the input unit from the dropdown menu:
Celsius
Fahrenheit
Kelvin
Convert Button
When the Convert button is clicked, the program reads the entered value and the selected unit.
It applies the correct mathematical formula to convert the temperature into the other two units.
Display Results
The converted temperatures are displayed clearly in the result section.
Reset Button
Clicking the Reset button clears the input field, resets the dropdown to Celsius, and removes the displayed results
